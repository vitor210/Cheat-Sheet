My very own Cheat Sheet

Here's my attempt at a cheat sheet for both table AND forms in HTML.


HTML Table References

Tag Name Explanation
<table> Table Used at the begin to create a table. Needs a closing </table> in the end.
<thead> Table Head Used for the titles of the collums.
<tbody> Table Body The actual "body" of the table, where the data will be.
<tfoot> Table Foot The bottom of the table (optional). Used to summarize the table
<tr> Table Rows Used to create rows of data
<td> Table Data Used inside <tr>. Create as many <td> as the number of collums to be used.
<th> Data in Table Head Used inside Table Head instead of <td>. Automatic formatation shows bold.
rowspan Row Span Use if a row will span more than 1 collum. Example: rowspan = "2"
colspan Collum Span Use if a column spans more than 1 row. Example: colspan = "2"

HTML Form References

Tag Name Explanation
<input> Input Used at the begining of every form command, can take different atributes shown below. Must have an "id" atribute to be connected to a label.
<label> Label Used in conjunction with <input> and can take different atributes. Must be connected to an input through a "for" atribute.
<input> type ="checkbox" Checkbox Used for a multiple choice. Must have a name and a value atribute.
<input> type ="radio" Radio Button Used for when you want to use a single choice. Must have a name and a value atribute.
<input> type ="text" Text Used for when you want to type a short sentence. Must have a name atribute.
<textarea> Textarea type Used for when you want to write several sentences. Must have a name atribute and how many rows and columns.
<input> type ="number" Number Used for when you want to type numbers. Must have a name atribute.
<input> type ="range" Number Range Used for when you want to type a range of numbers, from X to Y. Must have a name atribute, a min and max range defined, and step (how fast the bar moves).
